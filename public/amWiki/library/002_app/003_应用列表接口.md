## 接口详情 

### 请求地址
/app/list

### 请求类型
get

### 请求参数
| 参数名 | 类型 | 必填 | 描述 | 默认值 |
| --- | :---: | :---: | --- | --- |
| app_id | string | 否 | 应用id | - |
| type | string | 是 | 应用类型 0为安卓 1为IOS | 1 |


### 返回JSON示例
```javascript
{
    "code": 200,
    "msg": "请求成功",
    "data": [
        {
            "app_id": 1,    //应用id
            "name": "QQ",   //应用名
            "logo": "",     //应用图标
            "type": 1,      //应用类型 0为安卓 1为IOS
            "money": "1.20",    //奖励金额
            "num": 2,       //应用数量
        }
    ]
}
```
