## 接口详情 

### 请求地址
/app/list

### 请求类型
get

### 请求参数
| 参数名 | 类型 | 必填 | 描述 | 默认值 |
| --- | :---: | :---: | --- | --- |
| app_id | string | 否 | 应用id 获取应用详情时必传 | - |
| type | string | 是 | 应用类型 0为安卓 1为IOS | 1 |
| page | string | 是 | 页码 | 1 |
