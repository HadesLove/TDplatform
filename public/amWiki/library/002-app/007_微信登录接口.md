## 接口详情 

### 请求地址
/wechat/login

### 请求类型
any

### 请求参数
| 参数名 | 类型 | 必填 | 描述 | 默认值 |
| --- | :---: | :---: | --- | --- |
| code | string | 是 | 微信授权获取的code| - |
| appid | string | 是 | 微信公众号的appid| - |
| type | string | 是 | 用户类型| - |



### 返回JSON示例
```javascript

{
    "code": 200,
    "msg": "请求成功",
    "data": {
        'token': '', // 自动登录需要的token,
        'user_id': 1,   // 用户id
        'head_img': '', // 用户头像
        'nickname': 'xxxx', //用户昵称
    }
}

```
