## 接口详情 

### 请求地址
/admin/deal/pay

### 请求类型
get

### 请求参数
| 参数名 | 类型 | 必填 | 描述 | 默认值 |
| --- | :---: | :---: | --- | --- |
| withdraw_id | string | 是 | 提现明细列表的id | - |
| status | string | 是 | 提现状态  2为打款成功 3为打款失败 | - |
| note | string | 是 | 打款备注 | - |




### 返回JSON示例
```javascript
{
  "status": {
      "code": 200,
      "msg": "请求成功",
  }
  "data":  []
}
```
