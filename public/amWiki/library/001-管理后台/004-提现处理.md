## 接口详情 

### 请求地址
/admin/withdraw/list

### 请求类型
get

### 请求参数
| 参数名 | 类型 | 必填 | 描述 | 默认值 |
| --- | :---: | :---: | --- | --- |
| status | string | 是 | 提现状态 1为待提现 2为提现成功 | - |
| nickname | string | 否 | 用户昵称 | - |
| mobile | string | 否 | 用户手机号 | - |
| alipay | string | 否 | 提现账号 | - |
