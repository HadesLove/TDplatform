## 接口详情 

### 请求地址
/admin/app/add

### 请求类型
get

### 请求参数
| 参数名 | 类型 | 必填 | 描述 | 默认值 |
| --- | :---: | :---: | --- | --- |
| type | string | 是 | 应用类型 0为安卓 1为iOS | 0 |
| name | string | 是 | 应用名称 | - |
| logo | file | 是 | 应用logo | - |
| money | string | 是 | 应用奖励 | - |
| num | string | 是 | 应用份数 | - |
| rank | string | 否 | 搜索排名 | - |
| note | string | 否 | 备注 | - |
| pack_name | string | 否 | 包名 | - |


### 返回JSON示例

```javascript
{
  "code": 200,
  "msg": "添加成功",
  "data": []
}
```
